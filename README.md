# Drups.io documentation

Document built with [Sphinx](https://www.sphinx-doc.org).

## Installation

```shell
$ pip install -r docs/requirements.txt
```

## Usage

### Generate HTML from the docs

```shell
$ cd docs
$ make html
```

**Note**: You can see all available `Make` command by running `make help`
